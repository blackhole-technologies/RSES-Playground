# Project Vision: Embodied AI Assistant

## The Goal
Create a face-to-face conversational AI assistant that:
- Has full awareness of the filesystem through terminal access
- Can be spoken to naturally via voice
- Responds with voice and visual presence (avatar)
- Executes commands and shows results in real-time
- Organizes projects across multiple dimensions
- Compares AI coding styles (Claude vs ChatGPT vs Gemini)

## Evolution Phases

### Phase 1: Foundation (COMPLETE ✅)
- Multi-dimensional symlink organization
- Auto-link scripts with security
- Skip patterns for dependencies
- Agent-based workflow
- Documentation for context transfer

### Phase 2: Web Workbench (NEXT 📋)
- Browser-based UI
- Claude Code as backend (terminal commands)
- Filesystem tree visualization
- Symlink graph view
- Interactive command execution
- Real-time updates

### Phase 3: Voice + Avatar (FUTURE 🎯)
- Voice input (speak to Claude)
- Voice output (Claude speaks back)
- Avatar visualization (face-to-face)
- Lip-sync and expressions
- Natural conversation flow
- Spatial audio

## Why This Matters

- Natural human-AI interaction (voice, face-to-face)
- Transparent system awareness (sees filesystem, knows context)
- Multi-dimensional thinking (topic, type, AI tool, status)
- Persistent memory (documentation, git, autolinks)
- Extensible (new agents, new view dimensions)
