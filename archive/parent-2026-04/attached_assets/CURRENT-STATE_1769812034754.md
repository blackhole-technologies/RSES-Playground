## System Architecture

The Auto-Link system now operates on **two parallel tracks**:

| System | Status | Purpose |
|--------|--------|---------|
| **RSES** (bin/rses) | PRODUCTION | Primary symlink engine |
| Shell scripts (auto-link-*.sh) | DEPRECATED | Legacy, maintained for fallback |

**Use RSES for all new operations:**
```bash
bin/rses link          # Create symlinks
bin/rses link --dry-run # Preview changes
bin/rses status        # Show system state
bin/rses sets          # List defined sets
bin/rses rules         # List mapping rules
```

## Quick Reference

### Daily Usage
```bash
# Create/update all symlinks
bin/rses link

# Preview without changes
bin/rses link --dry-run

# Check system status
bin/rses status

# Start file watcher
bin/rses-watch --daemon

# Stop file watcher
bin/rses-watch --stop
```

### Configuration Editing
```bash
# Edit mappings
$EDITOR ~/.autolink.conf

# Apply changes
bin/rses link
```

### Debugging
```bash
# View defined sets
bin/rses sets

# View mapping rules
bin/rses rules

# Check audit log
tail -50 ~/.audit.log

# View watcher log
tail -50 ~/.rses-watch.log
```
